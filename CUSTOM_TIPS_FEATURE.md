# 自訂注意事項功能使用說明

## 功能概述

這個健身 APP 現在支援讓使用者為每個運動添加自己的個人注意事項，並可以永久儲存方便日後查看。

## 主要功能

### 1. 新增自訂注意事項
- 在運動詳情頁面，向下滾動到「? 我的注意事項」區域
- 在文字框中輸入您的個人注意事項
- 點擊「儲存注意事項」按鈕
- 您的注意事項會自動儲存到本地儲存空間

### 2. 管理已儲存的注意事項
- **查看**：已儲存的注意事項會顯示在藍色背景的區域中
- **編輯**：點擊「編輯」按鈕可以修改所有注意事項
- **刪除單條**：每條注意事項右側有「?」按鈕可以刪除
- **清除全部**：點擊「清除全部」可以刪除該運動的所有自訂注意事項

### 3. 注意事項管理頁面
- 在主頁面點擊「? 查看我的注意事項」按鈕
- 可以查看所有運動的自訂注意事項總覽
- 支援搜尋功能，可以搜尋運動名稱或注意事項內容
- 可以一次清除所有運動的注意事項

## 使用步驟

### 步驟 1：選擇運動
1. 在主頁面選擇身體部位（如：胸、腿等）
2. 在運動列表中選擇特定運動
3. 進入運動詳情頁面

### 步驟 2：新增注意事項
1. 向下滾動到「? 我的注意事項」區域
2. 在文字框中輸入您的注意事項
3. 點擊「儲存注意事項」按鈕
4. 您的注意事項會立即顯示在下方

### 步驟 3：管理注意事項
- **編輯**：點擊「編輯」按鈕，修改後點擊「儲存」
- **刪除**：點擊單條注意事項旁的「?」按鈕
- **清除全部**：點擊「清除全部」按鈕

### 步驟 4：查看所有注意事項
1. 返回主頁面
2. 點擊「? 查看我的注意事項」按鈕
3. 在管理頁面查看所有運動的注意事項
4. 使用搜尋功能快速找到特定內容

## 功能特點

### ? 資料安全
- 所有注意事項都儲存在瀏覽器的本地儲存空間
- 資料不會上傳到伺服器，保護您的隱私
- 清除瀏覽器資料會導致注意事項遺失

### ? 編輯功能
- 支援編輯已儲存的注意事項
- 可以刪除單條注意事項
- 可以一次清除所有注意事項

### ? 搜尋功能
- 在管理頁面可以搜尋運動名稱
- 可以搜尋注意事項內容
- 即時顯示搜尋結果

### ? 統計資訊
- 顯示總共有多少個運動有自訂注意事項
- 搜尋時顯示符合條件的數量

## 注意事項

1. **字數限制**：每條注意事項最多 500 字
2. **資料持久性**：注意事項會永久儲存，除非手動清除
3. **瀏覽器相容性**：需要支援 localStorage 的現代瀏覽器
4. **資料備份**：建議定期備份重要的注意事項

## 技術細節

- 使用 React Hooks 管理狀態
- 使用 localStorage API 進行資料持久化
- 支援響應式設計，適配各種螢幕尺寸
- 使用 Tailwind CSS 進行樣式設計

## 故障排除

### 問題：注意事項沒有儲存
**解決方案**：
- 檢查瀏覽器是否支援 localStorage
- 確認瀏覽器沒有禁用本地儲存
- 嘗試重新整理頁面

### 問題：無法編輯注意事項
**解決方案**：
- 確認已儲存了注意事項
- 點擊「編輯」按鈕後再進行修改
- 修改完成後點擊「儲存」按鈕

### 問題：搜尋功能無法使用
**解決方案**：
- 確認已輸入搜尋關鍵字
- 檢查關鍵字是否正確
- 嘗試清除搜尋框重新輸入
