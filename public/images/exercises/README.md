# 運動示意圖放置說明

## ? 圖片放置位置
將運動示意圖放在此資料夾中：`public/images/exercises/`

## ?? 需要的圖片檔案
根據程式碼，您需要準備以下圖片檔案：

### 胸部運動
- `pushup.png` - 伏地挺身
- `dumbbell_bench_press.png` - 躺臥推胸 - 啞鈴
- `diamond_pushup.png` - 鑽石伏地挺身

### 二頭肌運動
- `bicep_curl.png` - 啞鈴彎舉
- `hammer_curl.png` - 錘式彎舉

### 三頭肌運動
- `tricep_dip.png` - 三頭肌下壓
- `overhead_tricep_extension.png` - 過頭三頭肌伸展

### 肩膀運動
- `shoulder_press.png` - 肩推

### 背部運動
- `pull_up.png` - 引體向上

### 腹部運動
- `crunch.png` - 仰臥起坐

### 腿部運動
- `squat.png` - 深蹲

### 臀部運動
- `glute_bridge.png` - 臀橋

## ? 圖片規格建議
- **格式**：PNG（推薦）或 JPG
- **尺寸**：建議 300x200 像素或更大
- **檔案大小**：建議小於 500KB
- **內容**：清晰的運動動作示意圖

## ? 如何取得圖片
1. **免費圖片網站**：
   - Unsplash (unsplash.com)
   - Pexels (pexels.com)
   - Pixabay (pixabay.com)

2. **健身相關圖片**：
   - 搜尋關鍵字：fitness, exercise, workout
   - 例如：pushup exercise, dumbbell curl, squat exercise

3. **自製圖片**：
   - 拍攝自己的運動照片
   - 使用繪圖軟體製作示意圖

## ?? 注意事項
- 確保圖片版權合法
- 圖片命名必須與程式碼中的路徑完全一致
- 如果圖片無法載入，會顯示預設的佔位圖片
- 建議使用 PNG 格式以獲得更好的圖片品質
