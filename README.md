# 健身訓練 APP

一個使用 React 和 Tailwind CSS 建立的健身訓練應用程式。

## 功能特色

- **身體部位選擇**：首頁顯示可訓練的身體部位方塊
- **運動列表**：點擊身體部位後顯示該部位的訓練動作
- **運動詳情**：顯示運動的名稱、示意圖和注意事項
- **響應式設計**：支援各種螢幕尺寸
- **現代化 UI**：使用 Tailwind CSS 建立美觀的介面

## 技術架構

- **React 18**：前端框架
- **Tailwind CSS**：樣式框架
- **JavaScript**：程式語言

## 專案結構

```
src/
├── components/          # React 元件
│   ├── BodyPartGrid.js    # 身體部位選擇網格
│   ├── ExerciseList.js    # 運動列表
│   └── ExerciseDetail.js  # 運動詳情
├── data/               # 資料檔案
│   └── exerciseData.js    # 運動資料
├── App.js              # 主要應用程式元件
├── index.js            # 應用程式入口
└── index.css           # 全域樣式
```

## 安裝與執行

1. 安裝依賴項目：
```bash
npm install
```

2. 啟動開發伺服器：
```bash
npm start
```

3. 開啟瀏覽器前往 `http://localhost:3000`

## 使用方式

1. **選擇身體部位**：在首頁點擊想要訓練的身體部位
2. **選擇運動**：在運動列表中點擊想要學習的運動
3. **查看詳情**：查看運動的詳細說明和注意事項
4. **返回**：使用返回按鈕回到上一頁
